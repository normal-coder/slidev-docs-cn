VS Code 插件提供一些特性，这些特性可以帮你更好的组织幻灯片并且可以快速浏览。

### 特性 {#features}

- 在侧边面板中查看幻灯片
- 下一页 / 上一页按钮
- 对幻灯片重新排序
- 幻灯片折叠
- 将 Markdown 转为 HTML
